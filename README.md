# üé¨ Bookmyshow Console Application

A simple **C++ console-based ticket booking system** that simulates cinema seat reservations.  
This project demonstrates arrays, pointers, recursion, and user input handling while providing a practical seat-booking workflow.

---

## ‚ú® Features
- **View Seat Layout**: Displays the current seating arrangement with booked and available seats.
- **Book a Single Seat**: Reserve one seat by entering its row and column.
- **Book Block of Seats**: Book multiple adjacent seats in the same row.
- **Cancel Seat Booking**: Cancel a previously booked seat using row and column.
- **Cancel Block Booking**: Cancel multiple adjacent seats in the same row.
- **Display Available Seats Only**: Show only the seats that are currently free.
